# CPI Inflation Analysis

## Overview
This project analyzes CPI (Consumer Price Index) data using Microsoft Excel to identify inflation trends over time.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Charts & Dashboard

## Key Analysis
- Monthly CPI trend analysis
- Average CPI calculation
- Identification of peak inflation periods

## Insights
- Identified months with highest inflation
- Observed trend patterns across the dataset
